# Free Rein

## The DM-Less RPG

Complete Solo or Group RPGs without a DM. Remove game night DM stress and let everyone play the game! Where the players interpret the Oracle and drive the game forward.

## Paths

Be a Mage, a Warrrior, a Scoundrel, or an Accolyte.

## Features

* Roll The Oracle Dice for More Insight
* Generate Complete Unique Characters
* Generate A Unique Campaing Enemy
* Generate Unique Magical Items
* Customize Your Character At Any Time
